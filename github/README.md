# GitHub Resources

Terramate stacks for managing GitHub repositories with OpenTofu.

## Stacks

- **cddc39/** - Repositories under the cddc39 GitHub organization (currently
  empty)
- **dmikalova/** - Repositories under the dmikalova GitHub account

## Features

Each repository gets:

- Apache 2.0 license
- Branch ruleset protecting `main` (signed commits, linear history)
- Standard repository settings (no wiki, no projects, no issues)
